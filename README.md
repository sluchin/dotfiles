Quick start
-----------

  `git clone git://github.com/sluchin/dotfiles.git`  
  
  `cd dotfiles`  
  
  `./setup.sh`  
  
  `git submodule init`  
  
  `git submodule update`  
  
  `(install-el-get)`  
  
  `(el-get-install-all)`
