  - mew 6.5
  - w3m

Quick start
-----------

  `git clone git://github.com/sluchin/dotfiles.git`  
  `ln -s ~/dotfiles/.emacs.el ~/.emacs.el`  
  `ln -s ~/dotfiles/.emacs.d ~/.emacs.d`  
  `ln -s ~/dotfiles/.emacs.elc ~/.emacs.elc`
