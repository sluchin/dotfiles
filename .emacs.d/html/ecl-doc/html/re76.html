Name

mp:process-kill — Try to stop a running task.

Function

(mp:process-kill process)

process

An object of type mp:process.

Description

This function tries to stop a running task. Killing a process may fail if the task has disabled interrupts.

Example

Kill a task that is doing nothing.

(flet ((task-to-be-killed ()
         (loop (sleep 1)) ; Infinite loop
	 ))
  (let ((task (mp:process-run-function 'background #'task-to-be-killed)))
    (sleep 10)
    (mp:process-kill task)))
