Name

ecl_make_symbol — Find a lisp symbol

Function

cl_object ecl_make_symbol( const char * name, const char * package_name);

Description

This function finds or create a symbol in the given package. First of all, it tries to find the package named by package_name. If it does not exist, an error is signaled. Then, a symbol with the suppled name is searched in the given package. If the symbol exists, it is returned. If it does not exist, using INTERN.
