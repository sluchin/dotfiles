Name

ext:command-args — Original list of command line arguments.

Function

(ext:command-args )

returns

A list of strings.

Description

This function returns the list of command line arguments passed to either ECL or the program it was embedded in. The output is a list of strings and it corresponds to the argv vector in a C program. Typically, the first argument is the name of the program as it was invoked. You should not count on ths filename to be resolved.
