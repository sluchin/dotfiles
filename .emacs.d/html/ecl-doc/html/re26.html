Name

Number accessors — Unchecked conversion from Lisp types to C numbers

Functions and macros

cl_fixnum ecl_fixnum( cl_object n);

float ecl_single_float( cl_object n);

double ecl_double_float( cl_object n);

long double ecl_long_float( cl_object n);

Description

These functions and macros extract a C number from a Lisp object. They do not check the type of the Lisp object as they typically just access directly the value from a C structure.
