Name

Character case — C functions related to the character case

Functions and macros

bool ecl_upper_case_p( ecl_character c);

bool ecl_lower_case_p( ecl_character c);

bool ecl_both_case_p( ecl_character c);

ecl_character ecl_char_downcase( ecl_character c);

ecl_character ecl_char_upcase( ecl_character c);

Description

These functions check or change the case of a character. Note that in an Unicode context, the output of these functions might not be accurate (for instance when the uppercase character has two or more codepoints).
