13.1. Array limits

ECL arrays can have up to 64 dimensions. Common-Lisp constants related to arrays have the following values in ECL.

ConstantValue
array-rank-limit64
array-dimension-limitmost-positive-fixnum
array-total-size-limitarray-dimension-limit
