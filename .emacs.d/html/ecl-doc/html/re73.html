Name

mp:make-process — Create a new thread.

Function

(mp:make-process &key name initial-bindings)

name

A symbol to name the process. Processes can be unnamed and names need not be unique.

initial-bindings

The list of special variables which will be local to the new process. It defaults to T, which means copying all variables which are local to this process.

Description

This function creates a separate task, with a name, set of variable bindings and no function to run.

See also mp:process-run-function.
