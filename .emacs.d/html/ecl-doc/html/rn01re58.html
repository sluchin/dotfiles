Name

ext:*lisp-init-file-list* — ECL initialization files.

Constant

Type

A list of pathname designators.

Initial value

'("~/.ecl" "~/.eclrc")

Description

This variable contains the names of initialization files that are loaded by ECL or embedding programs. The loading of initialization files happens automatically in ECL unless invoked with the option -norc. Whether initialization files are loaded or not is controlled by the command line options rules, as described in ext:process-command-args.

Example

See ext:process-command-args.
