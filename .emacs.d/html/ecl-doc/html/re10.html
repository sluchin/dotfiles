Name

ecl_return0, ecl_return1, ... — Returning multiple values

Synopsis

ecl_return0( cl_env_ptr cl_env);

ecl_return1( cl_env_ptr cl_env, cl_object value1);

ecl_return2( cl_env_ptr cl_env, cl_object value1, cl_object value2);

ecl_return3( cl_env_ptr cl_env, cl_object value1, cl_object value2, cl_object value3);

Description

Returns N values from a C/C++ function in a way that a Common Lisp function can recognize and use them. The 0-th value is returned directly, while values 1 to N are stored in the Common Lisp environment cl_env. This macro has to be used from a function which returns an object of type cl_object.
