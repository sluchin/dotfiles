NameNicknameIn moduleDescription
COMMON-LISPCL, LISPECL coreMain Common Lisp package.

ASDF ASDFSystem definition file with ECL customizations.

In Table 9.1 we list all packages available in ECL. The nicknames are aliases for a package. Thus, system:symbol may be written as sys:symbol or si:symbol. The module field explains which library provides what package. For instance, the ASDF is obtained when loading the ASDF library with (require 'asdf); and the XLIB package when configuring and loading the CLX library.
