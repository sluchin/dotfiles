Python モジュールの配布¶

Release:2.7
Date:2011年 12月 26日

このドキュメントでは、 Python 配布ユーティリティ (Python Distribution Utilities, “Distutils”) について、モジュール開発者の視点に立ち、多くの人々がビルド/リリース/インストールの手間をほとんどかけずに Python モジュールや拡張モジュールを入手できるようにする方法について述べます。
