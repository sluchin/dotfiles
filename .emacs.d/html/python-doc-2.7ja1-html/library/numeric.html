9. 数値と数学モジュール¶

この章で解説されるモジュールは数値と数学関連の関数とデータ型を提供します。 numbers モジュールは数値型の抽象的階層を定義します。 math と cmath はさまざまな浮動小数点数および複素数向け数学関数を含みます。速度より10進法での正確さに興味があるユーザには、 decimal モジュールが真の10進表現をサポートしています。
