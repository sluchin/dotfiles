8. データ型¶

この章で解説されるモジュールは日付や時間、型が固定された配列、ヒープキュー、同期キュー、集合のような種々の特殊なデータ型を提供します。

Python にはその他にもいくつかの組み込みデータ型があります。特に、 dict 、 list 、 set (frozenset とともに古い sets モジュールを置き換えます)、そして tuple があります。 str クラスはバイナリデータや８ビットテキストを扱うことができ、 unicode クラスは Unicode テキストを扱うことができます。
