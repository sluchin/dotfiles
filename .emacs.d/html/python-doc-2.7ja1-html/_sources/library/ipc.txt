****************************
プロセス間通信とネットワーク
****************************

この章で解説されるモジュールは他のプロセスと通信するメカニズムを提供します。

いくつかのモジュール、たとえば :mod:`signal` や :mod:`subprocess` は同じマシン上での2つのプロセス間でだけ動作します。
他のモジュールはネットワークプロトコルをサポートし、2つかそれ以上のプロセスがマシンをまたいで通信するために利用できます。
