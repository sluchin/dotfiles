.. _allos:

************************************
汎用オペレーティングシステムサービス
************************************

本章に記述されたモジュールは、ファイルの取り扱いや時間計測のような
(ほぼ)すべてのオペレーティングシステムで利用可能な機能にインタフェースを提供します。
これらのインタフェースは、Unix もしくは C のインタフェースを基に作られますが、
ほとんどの他のシステムで同様に利用可能です。
概要を以下に記述します。
